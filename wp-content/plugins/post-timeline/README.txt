== Description ==

Post Timeline is the WordPress Plugin that creates Beautiful Timeline Page of your WordPress Posts, using this plugin user can create unlimited beautiful Vertical timelines, automatically create animated scroll navigation based on Tag or dates in ascending order.

<strong><a href="https://posttimeline.com/">DEMO OF POST TIMELINE</a></strong>


= Simple and Single Timeline =

Shortcode [post-timeline] to create Post Timeline on Page or Post.


= Header/Heading Post =

Shortcode [post-timeline header="55"]
Where 55 is the Post ID which you have created for Heading Post.


= Multiple Timelines =

To Create multiple timelines, Admin has to create multiple categories and assign posts to those categories, then use the shortcode below.

[post-timeline category_id="3"]

Where 3 is the category ID, all the posts associated with Category ID 3 will appear with the above shortcode.

[How to Create Multiple Timelines](https://posttimeline.com/docs/how-to-create-multiple-timelines/).

= How Navigation Works? =

Post Timeline Tags is an important tab of a timeline, through this tab Admin User can create Tags and associate their posts to those tags, once all the posts are assigned to those tags, the post timeline will collect all the tags and will use them in navigation.

[Tags for Navigation](https://posttimeline.com/docs/post-timeline-tags/).


= Other Post Types =

Use attribute ptl-post-type as shown below.
[post-timeline ptl-post-type="post"]

Where post in double quotation can be any post type.


For multiple post timeline, check the documentation, feel free to create a ticket in case of any problem or bug.

== Installation ==

Installation of this plugin is pretty easy.

1. Upload the extracted plugin folder to the `/wp-content/plugins/` directory of your WordPress installation, or upload it directly from your plugins section in the WordPress admin panel.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. That's it!

== Frequently Asked Questions ==

= How can I show Timeline by date? =

Add attribute [post-timeline ptl-type="date"], the timeline will appear with yearly navigation, the default navigation is Tag based.


= Getting Page Not Found on Read More button =

Goto Settings -> Permalinks -> Select "Post name" Name -> Click Save Changes

If Post name is already selected just click "Save Changes"


Create a ticket in support for further queries.


= Why my Post Timeline is showing on Right Side of page with scroll bar? =

Timeline works with 100% space on page, when you will make your container 100% with css, the timeline will appear properly.
https://posttimeline.com/
